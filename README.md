# 🌐 Digital Marketing Portfolio Website

A full-stack portfolio website with a responsive frontend and a functional contact form that stores messages in a MySQL database.

This project demonstrates frontend development, backend API creation, and database integration using Node.js, Express, and MySQL.

---

## 🚀 Features

- Responsive modern portfolio design  
- Campaign showcase section  
- Contact form with validation  
- Express.js REST API  
- MySQL database integration  
- Error handling (frontend & backend)

---

## 🛠 Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript (Fetch API)

**Backend**
- Node.js
- Express.js

**Database**
- MySQL
